Different input fields for forms

input field ATTRIBUTE:

#required(must be filled else warning)
#autocomplete(based on previous requests)
#autofocus(boolean attribute only one element can have autofocus as the element should automatically get focus whent thw page loads)
#formaction(specifies url of the file by overriding forms action attribute that will process the input control)
#formmethod(overrides method attribute of form)
#formnovalidate(for a specific input field)
#height and width
#ist(used to bind datalist created with input type)

#multple(user can enter multiple comma saperated values like email, pasasword etc)
